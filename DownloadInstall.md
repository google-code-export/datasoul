**Datasoul is written in Java and can run on Windows, Linux or Mac. Please find below the installation instructions for your system:**


---


## Windows ##
  * Datasoul requires Java 6 or later. You can download the latest version from http://www.java.com/
  * For video support Datasoul requires GStreamer, available [here](http://code.google.com/p/ossbuild/downloads/list?q=label:Featured-Yes+label:Package-Complete+label:License-LGPL). Download and install it to get video playback.
  * Download the Datasoul installer and run the installation.

> ### ![http://www.gstatic.com/codesite/ph/images/dl_arrow.gif](http://www.gstatic.com/codesite/ph/images/dl_arrow.gif) [Download Datasoul for Windows](http://code.google.com/p/datasoul/downloads/list?q=Featured+OpSys-Windows) ###


---


## Linux ##
  * You need Java 6 and GStreamer. You probably already have this if you are using an updated linux distribution.
  * Just get the package for Ubuntu/Debian or RPM and install it.

> ### ![http://www.gstatic.com/codesite/ph/images/dl_arrow.gif](http://www.gstatic.com/codesite/ph/images/dl_arrow.gif) [Download Datasoul for Linux](http://code.google.com/p/datasoul/downloads/list?q=Featured+OpSys-Linux) ###


---


## Mac ##
  * You need Java 6. You can get it using Software Update on MacOS 10.5 or later.
  * Download the application and unzip it unto the Application folder.
  * Currently, there is no installation package for GStramer on MacOS. Probably you can compile it yourself using macports, darwinports or fink, but we never did so.
  * Without GStreamer you can still use all features that does not require video playback.
